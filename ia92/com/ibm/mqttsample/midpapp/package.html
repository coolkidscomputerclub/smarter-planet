This package contains a J2ME MIDP-1.0 sample application. The MIDlet subscribes to a particular topic to receive data. Any data received is displayed on the devices UI and published (echoed) back to the broker on another topic.
There are three classes in this package.

  • Echo.java - This implements the application logic
  • WmqttGauge.java - This extends Gauge to make it continuously scroll
  • WmqttMgr.java - This customises the WMQTT API for the Echo application
